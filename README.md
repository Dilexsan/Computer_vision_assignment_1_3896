## Step-by-Step Setup

### 1. Create and Activate a Virtual Environment

Open a terminal in the project folder and run:

**For Windows:**
```bash
python -m venv venv
venv\Scripts\activate

```
### 2. Install packages

```bash

pip install -r requirements.txt
```

### 3. Run the Python code

```bash

python main.py
```
